🧾 FeedmobAI 报销系统

团队报销管理 · AI + OCR · 月度每人 ¥150 额度

📋 概览

  • 仅限 @feedmob.com 邮箱 Google 登录
  • 每人每月报销总额 ¥150，超出标记 ⚠️ 但仍可提交
  • 发票 OCR：火山 Ark vision 模型，支持 JPG/PNG/WebP/PDF
  • 外币自动换算 CNY（汇率来源：open.er-api.com）
  • 文件存储：Cloudflare R2，鉴权代理下载

🔑 获取 Token

登录 Web 页面 → 点击 🔑 Token 标签 → 复制 Bearer Token。

所有 API 请求头：Authorization: Bearer <token>

📡 API 端点

POST/api/ocr

上传发票 OCR 识别（multipart），返回 R2 存储链接 + 识别结果

curl -X POST {BASE}/api/ocr \
  -H "Authorization: Bearer TOKEN" \
  -F "invoices=@invoice.pdf"
POST/api/reimbursements/agent

智能体提交报销（JSON body）

curl -X POST {BASE}/api/reimbursements/agent \
  -H "Authorization: Bearer TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"amount":50,"description":"打车","attachments":[...]}'
POST/api/reimbursements

人工提交报销（multipart，含文件上传到 R2）

GET/api/reimbursements

查询列表。参数：?status=pending &limit=20

GET/api/reimbursements/:id

单条详情

PATCH/api/reimbursements/:id

修改报销单（仅 pending 状态）。Body: {"amount":60,"description":"更新"}

DELETE/api/reimbursements/:id

删除报销单

POST/api/reimbursements/:id/approve

审批通过

POST/api/reimbursements/:id/reject

审批拒绝。Body: {"reason":"发票不清晰"}

GET/api/dashboard

仪表盘统计

GET/api/exchange-rate

汇率查询。参数：?base=USD

GET/api/files/:key

鉴权下载 R2 文件（发票 PDF/图片）

🤖 MCP

端点：https://www.html.living/reimbursement/mcp（JSON-RPC）

Tools：submit_reimbursement, query_reimbursements, get_reimbursement_detail, approve_reimbursement, reject_reimbursement, get_dashboard_stats

⛓️ 约束

  • 月度总额：每人 ¥150
  • 文件格式：JPG / PNG / WebP / PDF
  • 文件大小：≤10MB
  • 仅支持 CNY（外币自动按当日汇率换算）
